MemoLife
¡Bienvenido a MemoLife! Esta aplicación te permite crear y gestionar tu perfil, añadir una foto de perfil y mantener una lista de tareas y notas con imágenes.

Tecnologías Utilizadas
MemoLife se ha desarrollado utilizando las siguientes tecnologías de React Native con Expo Go:

Funcionalidades

    Gestión de Perfil

Crear Perfil: Configura tu perfil con tu nombre, apellido y correo electrónico.

Foto de Perfil: Añade una foto de perfil desde tu cámara o galería.

Navegación: Usa los botones para navegar a tareas, notas o cerrar sesión para regresar a la pantalla de bienvenida.

    Gestión de Tareas

Crear Tareas: Añade tareas a tu lista de pendientes.

Marcar como Completadas: Marca tareas como completadas o no.

Eliminar Tareas: Elimina tareas de tu lista.

    Gestión de Notas

Crear Notas: Añade notas con títulos, cuerpos e imágenes.

Editar Notas: Modifica el contenido de tus notas existentes.

Eliminar Notas: Borra las notas que ya no necesites.

Navegación de la App

La app no está completamente lista, pero puedes empezar a usarla con los siguientes pasos:

Pantalla de Bienvenida: Usa el botón de inicio de sesión para ir a la pantalla de inicio de sesión.

Inicio de Sesión: Desde aquí, puedes pasar a la pantalla de perfil.

    Pantalla de Perfil:

Carga una imagen de perfil desde tu cámara o galería.

Observa el nombre, apellido y correo electrónico del usuario.

Usa los botones "task", "notes" o "cerrar sesión" para navegar.

    Pantalla de Tareas:

Crea y gestiona tu lista de tareas.

Marca tareas como completadas o elimínalas.

    Pantalla de Notas:

Crea nuevas notas o edita y elimina las existentes.

Visualiza el título, cuerpo e imágenes de tus notas.
